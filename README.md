# E-commerce Sales & Customer Behavior Analysis (SQL)

## 📌 Project Overview
This project focuses on analyzing transactional data from an e-commerce platform to derive actionable business insights. I performed customer segmentation, analyzed product category performance, and tracked user retention patterns.

## 🛠 Tech Stack
- **Database:** PostgreSQL
- **Tools:** pgAdmin 4
- **SQL Concepts:** Common Table Expressions (CTEs), Window Functions (LAG, OVER, PARTITION BY), Aggregate Functions, Case Logic.

## 📊 Key Analysis & Insights

### 1. Customer Segmentation (RFM-based)
- **Goal:** Categorize customers based on their total spending.
- **Business Value:** Helps the marketing team prioritize high-value customers for loyalty programs.

### 2. Product Category Performance
- **Goal:** Identify top-performing categories by revenue and customer reach.
- **Insight:** Calculated total sales volume and unique customer counts per category to find true "Best Sellers."

### 3. Retention & Loyalty Tracking
- **Goal:** Measure the time interval between consecutive orders for each customer.
- **Business Value:** Identifying churn risks by monitoring increasing gaps between purchases.

### 4. Marketing Channel & Device Efficiency
- **Goal:** Compare Average Order Value (AOV) across different devices and acquisition channels.
- **Insight:** Conducted benchmarking to see if Desktop users provide a higher AOV compared to Mobile users.
